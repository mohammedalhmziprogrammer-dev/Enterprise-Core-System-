# Enterprise Core System

مشروع Full-Stack باستخدام Django REST Framework و React.

## التقنيات المستخدمة

- Python / Django / Django REST Framework
- React.js / Vite
- PostgreSQL / SQLite

## طريقة التشغيل

### Backend

cd backend
python -m venv venv
pip install -r requirements.txt
python manage.py runserver

### Frontend

cd frontend
npm install
npm start
